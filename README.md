Performance Metrics: Visualizes core football metrics (Goals, Assists, Passing Accuracy, Shots on Target, Defensive Actions).

Team Comparison: Allows for side-by-side analysis of different teams across various leagues.

To view and interact with the dashboard, you need the free Tableau Reader software.

Download Tableau Reader: The file is a Tableau Package (.twbx), which requires Tableau Desktop or the free Tableau Reader to open. You can download Tableau Reader from Tableau's official website.

Open the Dashboard: Locate the football.twbx file in the cloned folder. Open the file using Tableau Reader to access the full interactive dashboard.
